# Morning demo

1. Start the local class system with `./scripts/start-demo.sh` from the Gymstant folder.
2. Open `http://education.localhost:8000` and sign in as `Administrator` / `admin`.
3. Start screen recording. Gymstant is now deliberately recordable.
4. Ask Gymstant: "Open the class software and show me the roster."
5. Let the Hermes handoff message remain visible; Gymstant will then retract to the working pill.
6. Click the working pill if you need to interrupt.
7. Ask for one fictional contact change, then demonstrate a monitored roster change that stops before final save.

All people and contact details in this environment are fictional.
