# Gymstant Workflow Watch Demo

## Goal
Show that Gymstant can watch a real workflow, protect proprietary information, turn the observation into an editable reusable manual, and remove a bad workflow cleanly.

## Before starting
1. Open the released Gymstant app.
2. Open a harmless sample workflow in the other application you will demonstrate. Use non-production or non-sensitive information.
3. In Gymstant, open Settings and confirm Screenshot privacy is set to **Pixelate sensitive content**.
4. Close Settings and open the Learn lane.

## Demo

1. Say: **“Watch this.”**
2. Point out that Gymstant has entered WATCHING THIS WORKFLOW mode and that no workflow is active until you deliberately demonstrate one.
3. Set the workflow title to: **Prepare a parent follow-up draft**.
4. In the sample application, open the parent record you prepared for the demonstration.
5. Return to Gymstant and click **Capture step**.
6. Set the first step label to: **Open the parent record**.
7. Set the first step note to: **Confirm the correct family before preparing any communication.**
8. In the sample application, open the communication or message area without sending anything.
9. Return to Gymstant and click **Capture step**.
10. Set the second step label to: **Open a new follow-up message**.
11. Set the second step note to: **Prepare the draft only; leave the final send action for a person.**
12. In the sample application, enter a short draft message but do not send it.
13. Return to Gymstant and click **Capture step**.
14. Set the third step label to: **Draft the follow-up message**.
15. Set the third step note to: **Check the recipient and wording, then stop before Send.**
16. Click **Save editable manual**.
17. Open the Review lane.
18. Point out the saved workflow, its captured-step count, and the **Remove** button.
19. Click **Remove** on the workflow you just created.
20. Point out that the workflow has disappeared from Review and Gymstant has recorded a local retirement note stating that it must not be used for future automation.
21. End the demo with: **“Gymstant observes a workflow once, protects what it sees, produces an editable manual, and lets us remove a workflow immediately when it is wrong or no longer needed.”**

## Expected outcome
- Three pixelated screenshots are stored locally as workflow evidence.
- The workflow is created in Review with editable steps and notes.
- The workflow is removed from the active queue.
- A local retirement record prevents it from being treated as an approved reusable workflow.
- No message is sent and no consequential action is completed.
